# React-Pizza_Shop : 


A simple and responsive pizza ordering app built with React. Users can browse through the menu, and place orders for their favorite pizzas.

## Tech Stack
- React
- Redux Toolkit
- Tailwind CSS
- React Router

  
## Features
- Menu with various pizza options
- Responsive design
